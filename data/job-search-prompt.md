# Job Search Prompt

## Updated Job Search Tasks

### Task 1 - Determine Industries from Work History
Identify the different industries based on your past work history.

### Task 2 - Retrieve Job Listings from Predefined URLs with Perth WA Filtering
Use the following job search platforms to retrieve listings for jobs located in Perth, Western Australia:
- [seek.com.au](https://www.seek.com.au)
- [indeed.com.au](https://www.indeed.com.au)
- [linkedin.com/jobs](https://www.linkedin.com/jobs)
- [cg-garage.com](https://www.cg-garage.com)
- [the-dots.com](https://www.the-dots.com)
- [artstation.com/jobs](https://www.artstation.com/jobs)
- [aria.org.au](https://www.aria.org.au)
- [polycount.org](https://www.polycount.org)
- [angel.co](https://www.angel.co)

Make sure to filter these job listings to those posted in the last 14 days.

### Task 3 - Generate Radar Chart Data
Prepare the data required for generating a radar chart to visualize job suitability.

### Task 4 - Extract Job Summaries
Extract short summaries for each relevant job listing gathered from the above sources. 

### Task 5 - Create Raw HTML Output with Metadata Sidebar
Format the extracted summaries and data into a raw HTML document, ensuring you include a metadata sidebar for additional context.
